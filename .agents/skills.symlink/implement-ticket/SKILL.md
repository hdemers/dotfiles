## Standard Operating Procedure:

1. Activate the skill jira-ticket.
2. Read the ticket
3. Transition the ticket to "In Dev".
4. Use the Plan tool to formulate a plan and seek approval from user.
5. Implement the plan.
6. Run tests
7. Activate the skill describe and add a commit message.
