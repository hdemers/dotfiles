Instructions to simplify code that was recently added or modified. 

# Standard Operating Procedure

1. Identify which revisions to analyze. If unsure stop and ask the user.
2. Generate the diff of those revisions
3. Do not re-invent the wheel. Search the following for functionalities to reuse:
    1. Existing codebase
    2. Standard library
    3. Third-party libraries
4. Produce a plan to simplify and refactor the code according to the following criteria:
   1. Idiomatic patterns
   2. Simple
   3. Elegant
   4. Reusable
